# Motorista Pro Uber

Suba ESTES ARQUIVOS na raiz do repositório.

Arquivos que precisam ficar visíveis no topo do repositório:
- index.html
- app.js
- styles.css
- manifest.webmanifest
- vercel.json
- package.json
- pasta icons

Se você enviar o ZIP para o GitHub sem extrair, a Vercel vai dar 404 porque ela não descompacta o arquivo automaticamente para servir o `index.html`.
